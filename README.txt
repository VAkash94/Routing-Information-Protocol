* The Routing Information Protocol Algorithm is implemented in ripv2.java

* Node class is used and it is given in Node.java

* Make sure Node.java is present in the same folder. You may compile and run only ripv2.java

* To run ripv2.java you can type in ‘java ripv2 filename’

* When one router is started it prints its table. Then give at least half a minute before starting another router.
